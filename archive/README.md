# Archive

This directory keeps the original Knowledge Graph build pipeline outside the
runtime application path.

The Streamlit demo and five-agent architecture use:

- `knowledge_graph/schema/`
- `knowledge_graph/visualizer.py`
- `scripts/init_neo4j.py`

The archived `layer1`, `layer2`, and `layer3` folders are retained only as
research/build references. Raw PDFs and generated search result files are not
included in the runtime repository.
